Pookie – AI Learning App for Kids

Description: Pookie is a fun and interactive mobile app that helps children learn using AI-powered tools like chatbots, voice recognition, grammar correction, and playful quizzes.

Features:

  • Interactive chatbot assistant for kids
  • Grammar and spelling correction for English and Arabic
  • Voice recognition for pronunciation
  • Colorful quiz modules with cute UI

Technologies Used:

Flutter Gemini API Firebase Hugging Face
