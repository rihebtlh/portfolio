🐾 Pookie – AI Learning App for Kids

A fun and interactive mobile app using AI tools like chatbots, voice recognition, and playful quizzes to help kids learn.

✨ Features

  • AI Chatbot for friendly Q&A (Gemini API)
  • Grammar correction (English & Arabic)
  • Pronunciation assistant with voice scoring
  • Interactive quizzes with cute animations
  • Bedtime stories, jokes, and fun facts

🛠 Technologies

Flutter · Dart · Gemini API · Hugging Face · Firebase · Speech-to-Text
